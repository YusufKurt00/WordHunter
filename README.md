# WordHunter 🔍

WordHunter, Selenium kullanarak bir web sitesindeki tüm bağlantıları tarar ve her sayfanın HTML içeriğinde belirli bir kelimenin geçip geçmediğini kontrol eder.

## Özellikler
- Belirtilen ana URL'deki tüm `href` linklerini toplar
- Her linkteki HTML içeriği taranır
- Hedef kelimeyi içeren sayfalar listelenir

## Kurulum
```bash
pip install selenium webdriver-manager
